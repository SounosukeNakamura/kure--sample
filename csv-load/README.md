# 呉市スポーツ少年団 登録状況管理

## 概要
呉市内のスポーツ少年団の登録状況を可視化・管理するダッシュボードアプリケーションです。
HTMLとJavaScriptのみで構築されており、サーバーやフレームワークは不要です。

## 機能

### データアップロード
- CSVファイルをブラウザから直接アップロード
- ファイル読み込み後、自動的にダッシュボードを表示

### サマリーカード
- 登録団数
- 総団員数
- スポーツ種目数
- 平均団員数

### グラフ表示
1. **少年団別団員数トップ10** - 棒グラフ
2. **スポーツ種目別団員数** - 比較棒グラフ
3. **少年団別団員分布** - 折れ線グラフ

### 詳細テーブル
- 全データを表形式で表示
- 団名またはスポーツ種目での検索機能
- ページネーション
- 表示件数変更（10/25/50/100件）

## データ項目

各少年団の以下の情報を記録します：

- **少年団名** - 団の名称
- **スポーツ種目** - 主に取り組んでいるスポーツ種目
- **活動頻度** - 活動を行っている回数（例：週3回、月4回など）
- **活動施設** - 利用している練習場所・体育館・グラウンドなどの施設名
- **所属団員数（男女別・世代別）**
  - 小学男子
  - 小学女子
  - 中学男子
  - 中学女子

## CSVフォーマット

```csv
少年団名,スポーツ種目,活動頻度,活動施設,小学男子,小学女子,中学男子,中学女子
呉野球少年団,野球,週3回,呉野球場,15,3,8,2
呉サッカー少年団,サッカー,週2回,呉中央グラウンド,12,8,10,7
呉バレーボール少年団,バレーボール,週2回,呉体育館,5,12,6,10
```

## 使用方法

1. `index.html` をブラウザで開く
2. 「少年団登録データCSVを選択」ボタンをクリック
3. CSVファイルを選択
4. 自動的にダッシュボードが表示されます
